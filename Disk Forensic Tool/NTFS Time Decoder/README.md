# DateTimeDecoder

## Overview
`DateTimeDecoder.py` is a Python script that decodes hexadecimal time values into readable date and time formats. It interprets hexadecimal strings representing the count of 100-nanosecond intervals since January 1, 1601, and converts them to a conventional date and time format.

## Requirements
- Python 3.x

## Installation
No additional installation is required as `DateTimeDecoder.py` utilizes Python's standard libraries.
